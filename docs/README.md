# Euleraph 项目文档

本目录包含 6 份“够用且可落地”的项目文档，用于支撑：需求对齐 → 设计落地 → 联调验收 → 上线回滚。

- 1️⃣ 需求卡（PRD-lite）：`requirements/01_prd_lite.md`
- 2️⃣ 技术方案（Design Doc Lite）：`design/01_design_lite.md`
- 3️⃣ 接口文档（API Spec）：`api/01_api_spec.md`
- 4️⃣ 数据文档（WiredTiger Schema & 口径）：`data/01_data_dictionary.md`
- 5️⃣ 测试清单（Checklist）：`test/01_test_checklist.md`
- 6️⃣ 发布与回滚 Runbook：`release/01_release_runbook.md`

建议在每个迭代中保持更新：
- 新增/变更接口 → 更新 API Spec
- 数据结构/口径变化 → 更新 Data Dictionary
- 上线方式/配置变更 → 更新 Runbook
